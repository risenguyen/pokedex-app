## Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/risenguyen/Pokedex-App
   ```
2. Navigate to the project folder:
   ```sh
   cd Pokedex-App
   ```
3. Install dependencies:
   ```sh
   npm i
   ```
4. Start the development server:
   ```sh
   npm run dev
   ```
